# Проект анализа настроений

## Описание

Этот проект предназначен для анализа настроений на основе текстовых отзывов. Мы используем машинное обучение для классификации отзывов как позитивных, негативных или нейтральных.

## Структура репозитория

- `data/`: Папка для хранения данных.
  - `raw/`: Сырые необработанные данные.
  - `processed/`: Обработанные данные.
  - `README.md`: Документация по данным.
- `src/`: Папка для исходного кода.
  - `preprocessing.py`: Код для предобработки данных.
  - `model.py`: Код для обучения модели.
  - `evaluate.py`: Код для оценки модели.
- `notebooks/`: Папка для Jupyter notebooks с анализом данных и моделированием.
- `docs/`: Папка для документации.
- `requirements.txt`: Список необходимых библиотек.
- `README.md`: Главный файл документации.
